## Processo de Criação (resumo rápido)

```
BRIEFING → ACERVO → PONTO DE VISTA → HEADLINE → ROTEIRO → DIREÇÃO DE ARTE → CAPTION → REVISÃO
```

### Checklist pré-publicação
- [ ] Headline para no scroll em 3s?
- [ ] 1 ideia por slide?
- [ ] Dados com fonte citada?
- [ ] Logo Asterisk no canto inferior direito?
- [ ] Último slide com CTA claro?
- [ ] Cor de acento consistente?
- [ ] O post tem ponto de vista — ou é só informação?
